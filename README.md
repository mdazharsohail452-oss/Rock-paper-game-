# Rock Paper Scissors Game – Best of 5 Edition
This project is a classic Rock Paper Scissors game built with HTML, CSS, and JavaScript, featuring a "Best of 5" match system. The first player (human or computer) to win 3 out of 5 rounds is declared the winner of the match.

Features
Play classic Rock Paper Scissors against the computer
"Best of 5" match – first to reach 3 wins is the overall winner
Automatic score tracking and result display after each round
Responsive, user-friendly interface compatible with desktops and mobiles

How to Play

Start the game in your web browser.
Select Rock, Paper, or Scissors to play a round.
The computer chooses its move automatically.
The winner of the round is shown, and scores are updated.
If either player reaches 3 wins, the match ends and that player is declared the overall winner.
Restart to play another best of 5 series.

Technologies Used
HTML for structure
CSS for design
JavaScript for interactivity, score keeping, and winner determination
